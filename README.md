Készítsen egy web-alkalmazást a következő feladatok megvalósítására:

1.	Főoldal menü (index.html): Mutassa be az alkalmazást látványos weboldalon.		(2 pont)
Fejlécbe írassa ki H1-es címsorral: Web programozás-1 Előadás Házi feladat
Láblécbe írja be a készítők nevét és Neptun kódját.
2.	JavaScript menü (javascript.html): 								(2 pont)
Készítsen egy CRUD alkalmazást JavaScript-el, amihez használja fel a választott adatbázis egyik fájljának adatait. Az alkalmazás adatait egy tömbben tárolja.
3.	React menü (react.html): 									(2 pont)
Készítsen egy CRUD alkalmazást React-el, amihez használja fel a választott adatbázis egyik fájljának adatait. 
4.	SPA menü (spa.html): 										(3 pont)
Készítsen egy Egyoldalas alkalmazást (Single page application, SPA) 2 menüvel. A két menüpontnál valósítson meg 1-1 szabadon választott React alkalmazást, amelyeknek a nehézségi foka legyen hasonló, mint az órán bemutatott Calculator, Tic-Tac-Toe játék alkalmazások nehézségi foka. 
Mindkettőben használjon komponenseket és useState állapotváltozókat.
A dokumentációban adja meg, hogy honnan használta fel a két React alkalmazást.
A GitHub-on készítsen egy react mappát, amibe tegye fel a két alkalmazás src és dist mappáinak a tartalmát.
5.	Fetch API menü (fetchapi.html):								(4 pont)
Készítsen egy CRUD alkalmazást JavaScript-el és Fetch API-val, amihez használja fel a választott adatbázis egyik fájljának adatait. Az alkalmazás adatait a szerver oldalon adatbázisban tárolja.
6.	Axios menü (axios.html)									(4 pont)
Készítsen egy CRUD alkalmazást React-el és Axios-al, amihez használja fel a választott adatbázis egyik fájljának adatait. Az alkalmazás adatait a szerver oldalon adatbázisban tárolja.
7.	OOJS menü (oojs.html)										(3 pont)
Készítsen egy szabadon választott Objektumorientált JavaScript grafikus alkalmazást. Alkalmazza a következőket: class, constructor, metódusok, document.body.appendChild, extends, super
